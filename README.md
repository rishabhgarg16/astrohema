# Astro Hema

Professional astrology consultation website for Astro Hema.

## Project Structure

- `/web` - Next.js website
- `/design-references` - Design inspiration screenshots

## Quick Start

```bash
cd web
npm install
npm run dev
```

Visit http://localhost:3000

## Deployment

The website is ready to deploy to Vercel. Simply connect your GitHub repository to Vercel and it will auto-deploy.

See `/web/README.md` for detailed documentation.
